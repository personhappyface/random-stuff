1.4 Whats an HTML Tag

Learning HTML means understanding how the different tags will effect, or manipulate, text.
No one memorizes all the available HTML tags, that's what google is for.

You just need to understand how HTML gets displayed and parent - child relationships (I know that sounds weird).

First off, HTML doesn't try to trick you.
They do their best to be self-explanitory.

For example, Look at the code below.
What do you think <center> tags do? - Click on it to find out.

<center>How will I be displayed in a web page</center>
How will I get displayed in a web page

Text between center tags will become centered on a website. Easy, right!

Displaying HTML

The Browser reads HTML code from the top to bottom.
Whatever gets coded first (top) will be displayed ABOVE the tags coded beneath (bottom).

Parent - Child Tags

If you create an HTML tag between an existing HTML tag, they have a parent - child relationship.
That's how it works in HTML.

Notice the HTML tag opens, but doesn't close until both head and body have closed.

For example, the head and body tags are both children of the main html tag.
Which also means the head and body are siblings.

It's important to understand this, because when you read online about HTML they will discuss it in this way.

This relationship makes it so multiple HTML tags can effect the same text.

Make sure you notice that HTML tags need to close in the reverse order that they opened.
For example: (Click on the text below!)


<center> <strong> <u> <small> Hello World! </small> </u> </strong></center>
Hello World!


The text 'Hello World' is effected by Five different tags.
The center tag is the parent in this example.


Remember, HTML is a markup language. Any text between an opening and closing tag will be displayed based on what HTML tag(s) surround it.

The best way to find out how HTML tags will display content on a website is to test them out!

You'll be doing that very soon. You'll have a list of HTML tags to code into your Text Editor and then see how it gets displayed in the browser!
